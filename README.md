# MovieLookup

## Project Description

This project allows you to:

1. Search for movies
2. Go through search results with pagination
3. Add movies to watchlist
4. Delete movies from watchlists

### Features

- Search movies by Title
  - Search Suggestions Menu
  - Search Results Page
- Add movies the watchlist
- Remove movies from the watchlist

## How to run the project

1. Download the repo to your machine
2. Make sure you have the `APIKEY` generated from the OpenWeatherMap
3. Create a `.env` file at the root of the `/project-folder` then add this line into it:

```
VITE_OMDB_APIKEY=paste-your-api-key-here
```

4. Run this command in your terminal `npm run dev`
5. The terminal shows which port your localhost runs
6. Enjoy :D
